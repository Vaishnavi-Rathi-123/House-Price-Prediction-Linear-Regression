# House Price Prediction using Linear Regression

## Problem Statement
HomeVista Properties wants to automate house price prediction using machine learning
based on physical, location, and construction features.

## Dataset Description
The dataset contains historical house sale records with features like:
- Lot Area
- Zoning
- Overall Condition
- Year Built
- Basement Area
- Sale Price (Target)
